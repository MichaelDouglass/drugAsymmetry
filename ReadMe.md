## Exploring Asymmetric Application of Drug Applications
### Hinczewski Lab -- Case Western Reserve University
### Michael Douglass

This repository explores secondary effects of drug applications as reported in
"Steering Evolution with Sequential Therapy to Prevent the Emergence of Bacterial
Antibiotic Resistance" (doi: 10.1371/journal.pcbi.1004493).

Specifically, we are investigating the asymmetric weight of steering drugs applications.
